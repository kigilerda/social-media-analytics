# social-media-analytics

Пет-проект: построение пайплайна **классификации “популярности” постов по тексту**.  
Цель — предсказывать, станет ли пост **популярным**

---

## Данные и постановка задачи

**Источник:** датасет постов с канала из телеграмма с полями `text`, `views`, `reactions`, `date`.

1) Сначала рассчитывается вовлеченность по формуле: engagement = reactions / views
2) Затем формируется бинарный таргет:

target = 1, если engagement >= Q0.75 (верхний квартиль)

target = 0 иначе

Таким образом, “популярные” посты — примерно верхние 25% по engagement.

## Результаты (test)

1) модель находит около 60% действительно популярных постов (Recall=0.603),
2) среди предсказанных “популярных” около 58% действительно популярные (Precision=0.585).

Что, по-хорошему, бы улучшить:
1) подбор порога классификации под бизнес-цель (увеличить Recall или Precision),
2) добавление PR-AUC и калибровки вероятностей.
